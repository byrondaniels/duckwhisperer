Plume

Private voice typing for your Mac. Press Option+Space, speak, press Option+Space again, and Plume pastes polished text where your cursor already is.

Install

  1. Drag Plume.app into Applications.
  2. Open Plume from Applications.
  3. Open the menu-bar icon and choose Finish Setup....
  4. Grant Microphone and Paste-Back permissions when macOS asks.
  5. Choose Try It Here in the setup window for a safe first dictation.

Use It

  • Option+Space: start or stop recording.
  • Escape: cancel without pasting.
  • Try It Here...: test Plume without leaving the app.
  • Undo Last Paste: quickly recover if text lands somewhere unexpected.
  • Time Saved: see how much typing Plume has replaced.

Customize

  • Writing Mode: choose clean email, notes, bullets, raw dictation, and more.
  • Saved Words...: teach Plume your names, brands, and phrases.
  • Settings: adjust capitalization, per-app defaults, help, and permissions.
  • Settings -> Advanced: language, model, translation, and fun output modes.
  • Export Support Bundle...: create diagnostics without transcript history.

Privacy

English transcription runs locally. Optional non-English, translation, and enhanced style assets download only after you approve them.

Your runtime files live in ~/Library/Application Support/Plume.

Permissions

If Plume shows a transcript window instead of pasting, your text is copied and safe. Use Paste Again, Copy, Try In Plume, or Fix Permission.

Open Finish Setup... for the paste-back dashboard, or go to System Settings -> Privacy & Security -> Accessibility and enable Plume.

Optional Downloads

Non-English input, local translation, and Enhanced Robot mode are installed from Settings -> Advanced -> Speed & Accuracy only when selected.

The packaged app includes Best Accuracy English dictation so a new user can start without a terminal.
